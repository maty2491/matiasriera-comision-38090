Componente Funcionales
    - Importar react(opcionol)
    - Crear una función
    - El nombre de la función debe comenzar con mayúscula
    - La función debe retornar algo
    - Exportar el Componente

Todo Componente puede tener 
    - props
    - states
    - Ciclo de vida
    - Metodos del ciclo de vida

Ciclo de vida y Metodos

    - mount
        -constructor
        -render
        -componentDidMount
    
    - update
        -constructor
        -render
        -componentDidUpdate

    - unmount
        -componentWillUnmount

      Get: obtener informacion. Ejemplo consulta a API que tieen productos.
      Post: Cargar informacion enviar informacion al backend y por lo tanto a una base de datos.
      Delete: Borrar. Ejemplo un delete del usuario para que no le genere de SPAM.
      PUT: Se utiliza para modificaciones. Ejemplo para modificar un usuario.
      PATH: Te permite remplazar todos los datos de un usuario por ejemplo.
